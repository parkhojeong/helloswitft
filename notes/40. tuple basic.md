* tuple
- combination of related data
```swift
let ios = ("Swift", "5")
```
- access by index
```swift
ios.0
ios.1
```
- named tuple
```swift
let ios = (language: "Swift", version: "5")
ios.language
ios.version
```
- tuple decomposition
```swift
let (language, version) = ios
```
- tuple value comparison
```swift
("Swift", "5") == ("Swift", "5")
```
