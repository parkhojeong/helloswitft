* type comment
```swift
let a: Int = 1
```

* type inference
```swift
var name = "john"
```

* type safety
- swift allows operation between same type only
```swift
let a = 1
let b = 1.5
// a + b // not allowed 
```

* type casting
- type casting can make loss of data
```swift
Int("123") // 123(Int)
Int("abc") // nil
Int(123.45) // 123
```
