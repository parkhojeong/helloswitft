* switch statement
- expression/varaible matching condition
```swift
var choice = "rock"
switch choice {
    case "rock": print("rock")
    case "paper": print("paper")
    default: print("default")
}
```

* switch characters
1) case can have one or more matching values
```swift
switch ch {
    case "a", "b": print("a or b")
    default: print("none of the above")
}
```
2) exhaustive switch: all cases must be covered
3) each case must have one or more statements
- break can be used 

- fallthrough: execution continues next case
```swift
switch num {
    case 1: 
        print("one") 
        fallthrough
    case 2: 
        print("two") 
        fallthrough
    default: print("default")
}
```

- range operator, pattern matching operator
```swift
var num = 25
0...50 ~= num
```
