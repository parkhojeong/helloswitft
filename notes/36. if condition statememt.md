* programming basic mechanism
all programming consist of three logics
1) sequence: code execute from top to bottom
2) condition
3) iteration

* if condition statement
- condition: boolean expression
```swift
if condition {
    // code
}

// example
if number < 6 {
    // code
}
```
- condition order is important
  - close and wide from top to bottom
```swift
if n > 50 {
  // code
} else if n > 70 { // never execute
  // code
}

if n > 70 {
  // code
} else if n > 50 {
  // execute
}
```
- nested if statement is allowed
```swift
if n > 50 {
  if n > 70 {
    // code
  }
}
```
