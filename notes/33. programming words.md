* words
1) keyword: reserved word
- ex. var, let, if, for, in, etc
2) literals: data value itself(Int literal, etc)
- ex. "hello", 6, 3.14
3) identifier
- variable/constant name
- function name
- user-defined type name
4) token
- code minimum unit(identifier, keyword, punctuation, operator, literal)
5) expression
- code unit that can be evaluated to a value
```swift
var n = 5 // statement
17 // expression
n // expression
n + 6 // expression
n < 5 // expression
```
6) statement
- code unit that can be executed
```swift
var n2 = 3
print(n)
print("hello")
```
