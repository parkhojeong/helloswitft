* 0 and 1
- computer can recognize only 0, 1

* 10 digits, 2 digits
10 digits: unit number is 10(0~9)
2 digits: unit number is 2(0~1)

* 10 digits to 2 digits
- divide by 2 until Quotient is 1

   1 .. 0 
-> 101(2)

2  6

* data in memory
- consist of 0,1
- 1 bit: 0 or 1
- minimum data unit: 8 bit (1byte)

* data unit
- 1byte = 8bit
- 2^10 bytes : 1K Bytes
- 2^20 bytes : 1M Bytes
- 2^30 bytes : 1G Bytes
- 2^40 bytes : 1T Bytes

* 10 digits, 2digits, 16 digits
- 10 digits: 0~10
- 2 digits(0b): 0~1
- 16 digits(0x): 0~F

* 2 digits and 16 digits
- 4bits: nibble, 8bits: byte
- 16 digits has 16 cases(4bit cases)
  - ex. 0b 1111 0000 -> 0x F 0
  - ex. 0b 0000 1111 -> 0x 0 F  
- 2 digits <-> 16 digits: easy transformation

* notes
- 32 bit computer: cpu can handle 32 bit -> 2^32 bit -> in memory 2^32 Bytes(4GBytes)
  - cpu register can have 32bit
- 64 bit computer: 2^64

1. computer consist of 0 and 1
