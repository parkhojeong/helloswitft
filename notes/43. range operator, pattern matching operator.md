* range operator
- 1...10
- rules
  - descending is not allowed
  - 1...10.1 is allowed
- usage
  - pattern matching in a switch clause
  - for loop(integer range)
  - with array subscript
- type
  - closed range operator
    - ClosedRange<>, PartialRangeFrom<>, PartialRangeThrough<> 
    - ex. 1...10, ...10, 1...
  - half-open range operator
    - Range<>, PartialRangeUpTo<> 
    - ex. 1..<10, ..<10


```swift
// for loop
for i in 1...10 {
  print(i)
}
// array subscript
names[...2]
// pattern matching
switch num {
    case 10...19: print("10~19")
    default:
}
```
