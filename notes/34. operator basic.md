* operator basic
1. unary operator: without space
- prefix operator
- postfix operator
ex. !a
2. binary operator: with one space both sides
- infix operator
ex. a + b
3. ternary operator: with one space between two operators
ex. a ? b : c

* basic operator
- assignment operator: =
- arithmetic operator
  - +, - (unary possible)
  - *
  - /
    - Int/Int: quotient. (10/3 = 3)
    - Double/Double: quotient and remainder(10.0/3.0 = 3.33...) 
  - %: only for Int

* operation caution
- two operands must be the same type
